# Excel 自动填表测试样例：30 个空缺

这个样例用于测试 agent 是否能从一个 Excel 数据源读取数据，并填写到另一个 Excel 模板中。

## 文件

| 文件 | 用途 |
| --- | --- |
| `01_source_esg_metrics_2025.xlsx` | 源数据表，包含 30 个 ESG 指标的填报值 |
| `02_target_esg_template_30_blanks.xlsx` | 待填模板，`填报模板!C4:C33` 共 30 个空缺 |
| `03_expected_filled_result.xlsx` | 标准答案，用于比对 agent 输出 |

## 推荐测试方式

在网页中上传这两个文件：

1. `01_source_esg_metrics_2025.xlsx`
2. `02_target_esg_template_30_blanks.xlsx`

然后输入任务：

```text
请读取 01_source_esg_metrics_2025.xlsx 的“数据源”工作表，
根据“指标编码”或“ESG指标”匹配 02_target_esg_template_30_blanks.xlsx 的“填报模板”工作表，
把源表中的“填报值”填写到目标模板 C4:C33 的 30 个空缺中。
不要修改已有的指标编码、ESG指标名称和单位。
完成后请导出填写后的 Excel，并生成填写报告，记录每个填写值、目标单元格和数据来源。
```

## 预期结果

- 应填写 30 个空缺。
- 目标范围：`填报模板!C4:C33`。
- 每一行优先按 `指标编码` 匹配，例如 `E-001`、`S-001`、`G-001`。
- 如果编码无法匹配，再按 `ESG指标` 名称匹配。
- 输出结果可以和 `03_expected_filled_result.xlsx` 对比。

## 关键检查点

| 检查点 | 预期 |
| --- | --- |
| 空缺数量 | 30 |
| 写入列 | `填报值` |
| 写入范围 | `C4:C33` |
| 匹配依据 | `指标编码` 或 `ESG指标` |
| 不应覆盖 | `指标编码`、`ESG指标`、`单位` |
| 报告要求 | 记录填写值、目标单元格、数据来源 |
