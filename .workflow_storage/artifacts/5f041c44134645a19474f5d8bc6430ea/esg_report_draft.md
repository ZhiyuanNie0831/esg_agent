# 2025年度ESG补充说明（审阅稿）

## 一、报告说明

本补充说明依据已提供的《01_source_esg_metrics_2025.xlsx》及《02_target_esg_template_30_blanks.xlsx》编制，覆盖环境、社会及治理三大维度的30项ESG指标。报告期间主要为2025年度累计或截至2025年12月31日。披露内容参考GRI Standards，并结合ISSB IFRS S1 / IFRS S2关于可持续发展及气候相关信息披露的关注方向进行组织。

本报告主体名称、组织边界、业务范围、合并口径、是否经第三方鉴证等信息，材料中暂未提供，均列为“待补充”。本稿仅作为客户审阅用ESG报告初稿，所有关键数据均来源于已上传表格材料。

## 二、管理层摘要

2025年度，公司围绕能源使用、温室气体排放、资源管理、员工发展、职业健康安全、供应链责任、董事会治理及商业道德等议题开展ESG数据统计。披露矩阵显示，GRI气候与排放、能源与转型、员工发展与培训、供应链与人权、董事会与治理架构、商业道德与合规六项主题均已有材料覆盖。

环境方面，年度总用电量为1,285,000 kWh，可再生电力采购量为522,000 kWh；范围一温室气体排放为1,840.6 tCO2e，范围二温室气体排放为2,965.4 tCO2e。公司2025年度环保合规处罚次数为0次，环保投入金额为860.5万元。

社会方面，截至2025年12月31日，公司员工总数为2,380人，女性员工占比43.8%，女性管理者占比36.5%。2025年度员工培训总小时数为68,420小时，人均培训小时数为28.7小时，员工流失率为7.9%。职业健康安全方面，工伤事故次数为3次，损失工时事故率为0.18次/20万工时。

治理方面，2025年度董事会审议ESG议题6次，独立董事占比42.9%，董事会女性成员占比28.6%。商业道德培训覆盖率为98.6%，反腐败举报处理完成率为100%，重大合规违规事件数量为0件，信息安全重大事件数量为0件。

## 七、数据缺口与后续动作

1. **报告边界待补充**：需明确报告主体名称、组织边界、业务范围、覆盖地区、合并口径及是否包含子公司或供应链环节。  
2. **核算口径待补充**：温室气体排放需补充核算标准、排放因子来源、基准年、计算方法及是否覆盖范围三排放。  
3. **目标与管理措施待补充**：材料中暂未提供节能、减排、节水、废弃物减量、员工发展、供应链责任及治理合规方面的年度目标和中长期目标。  
4. **管理制度待补充**：需补充环境管理制度、职业健康安全制度、商业道德政策、反腐败制度、信息安全政策、供应商行为准则文本等。  
5. **绩效解释待补充**：建议对关键指标变化原因、年度重点项目、改进措施及下一年度计划进行管理层说明。  
6. **鉴证情况待补充**：如公司计划发布正式ESG报告，应明确是否进行第三方鉴证及鉴证范围。
